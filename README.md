# 🐍 SnakeGame (Java)

Un semplice gioco Snake sviluppato in Java.  
Controlla il serpente, mangia il cibo e cerca di ottenere il punteggio più alto senza colpire i bordi o te stesso!

---

## 🎮 Caratteristiche

- Movimento fluido del serpente
- Sistema di punteggio
- Crescita del serpente quando mangia il cibo
- Rilevamento collisioni (pareti e corpo)
- Interfaccia semplice e leggera

---

## 🖥️ Tecnologie utilizzate

- Java
- Swing / AWT (per la grafica)
